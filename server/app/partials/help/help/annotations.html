The “APPRIS annotations” section shows the scores of the APPRIS modules.

No. Functional Residues

firestar predicts functionally important residues. The box shows the total number of functional residues detected by firestar for each transcript.

Tertiary Structure Score

Matador3D (and Matador3Dv2) analyze protein structural information for each isoform. For both, protein sequences from the same gene are mapped onto 3D structures by running BLAST against the PDB, but depending on whether or not we have genomic information, the score is calculated in one of two ways.

When we made use of genomic coordinates, the score is based on the number of exons that can be mapped to structural homologues. When we don't know the genomic coordinates, the score is the sum of bit-scores in PDB alignment.

This score is only comparable for transcripts within the same gene and should not be used to compare between genes.

Conservation score (vertebrates)

CORSAIR carries out BLAST searches against vertebrates to determine the most likely principal isoform. The score approximates to the number of species that have isoforms that can be aligned to each tested isoform without introducing gaps.

Domains score

SPADE identifies the functional domains present in a transcript and detects those domains that are damaged (not whole). The number represents the sum of bitscores in Pfam alignment.

No. Transmembrane Helices

THUMP makes unanimous predictions of trans-membrane helices. The score shows the number of trans-membrane helices predicted by THUMP for each variant.

Signal Peptide Sequences and Mitochondrial Signal Sequences

CRASH predicts the presence and location of signal peptides and cleavage sites in amino acid sequences using an especially reliable form of SignalP and TargetP. Isoforms that are predicted to have reliable signal sequences are marked.

No. Mapping Peptides

Proteomic evidence only for the human genome (GENCODE gene set).

PROTEO collects peptides from seven separate MS sources. Two came from large-scale proteomics databases, PeptideAtlas and NIST. Another four datasets were recently published large-scale MS experiments. For all six datasets the starting point was the list of peptides provided by the authors or databases. We generated the final set of peptides (referred to as “CNIO") in house from an X!Tandem search against spectra from the GPM and PeptideAtlas databases, following the protocol set out in Ezkurdia et al with a false discovery rate of 0.1%. These seven studies cover a wide range of search engines, tissues and cell types.

In order to improve reliability the peptides from each of these studies were filtered, eliminating non-tryptic and semi-tryptic peptides and peptides containing missed cleavages. For those studies where it was possible we considered only peptides identified by multiple search engines.

Normalized Functional Importance

TRIFID is a machine learning classifier based on APPRIS and external database input which we have found to be a reliable indicator of isoform functional importance.

The normalized TRIFID score is shown if TRIFID predictions are available for the given isoform.

For more information about this method, consult the TRIFID GitLab repository.
